## Descipción del proyecto

El siguiente proyecto es desarrollado para el establecimiento con el mismo nombre, la cuál tendrá como principal función la de tomar las ordenes de los clientes y cuando estos finalizen esta labor, la orden será enviada a los cocineros (as), así como al cajero que se encuentre encargado del comercio en ese momento, para que estos puedan comenzar comenzar con la preparación de los productos que en ese momento los clientes deseén consumir.
Al finalizar la estadía en el establecimiento el mesero debera realizar la confirmación de la orden para que el cajero pueda capturar los productos consumidos por el cliente y generé el ticket para el cliente.

## Descripción de la Base de Datos

Se hace uso de una Base de Datos MongoDB que es una de tipo no SQL, esta contine una base de datos llamado rcgb como acronimo de Rompiendo Géneros Coffee and Beer. Dentro de esta se tienen 4 colecciones llamadas employees, products, clients y orders los cuales serán usados para almacenar los datos de los empleados, productos, clientes y ordenes, respectivamente
